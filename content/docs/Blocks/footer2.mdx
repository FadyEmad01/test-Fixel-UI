### Install dependencies

This component uses the `cn` utility for class merging. Make sure you have it set up:

    ```tsx title="lib/utils.ts"
    import { clsx, type ClassValue } from "clsx"
    import { twMerge } from "tailwind-merge"

    export function cn(...inputs: ClassValue[]) {
      return twMerge(clsx(inputs))
    }
    ```

## Customization

### Change Brand Name and Logo

Modify the brand section to match your branding:

```tsx
<div className="flex items-center gap-2">
  <div className="flex size-8 items-center justify-center rounded-lg bg-primary">
    <span className="text-lg font-bold text-primary-foreground">YB</span>
  </div>
  <span className="text-lg font-semibold">Your Brand</span>
</div>
```

### Update Navigation Sections

Customize the footer sections by modifying the `footerSections` array:

```tsx
const footerSections: FooterSection[] = [
  {
    title: "Your Section",
    links: [
      { label: "Link 1", href: "/link1" },
      { label: "Link 2", href: "/link2" },
    ],
  },
  // Add more sections as needed
];
```

### Add or Remove Social Links

The footer includes GitHub, Twitter, and Discord icons. You can add or remove social links as needed. Replace the SVG icons with your preferred social platforms.

## Features

- **Responsive Design** - Adapts from 1 to 5 columns based on screen size
- **Brand Section** - Logo, description, and social links
- **Navigation Sections** - Up to 4 customizable navigation groups
- **Social Media Links** - GitHub, Twitter, Discord with SVG icons
- **Legal Links** - Privacy policy and terms of service in bottom bar
- **Accessible** - Proper ARIA labels and semantic HTML
- **Dark Mode Support** - Uses Tailwind's dark mode classes

## Props

| Prop      | Type   | Default | Description                                           |
| --------- | ------ | ------- | ----------------------------------------------------- |
| className | string | -       | Additional CSS classes to apply to the footer element |
